De globale instellingen voor referentiepunten die van toepassing zijn op alle pagina‑elementen.

Instellingen die het gedrag van het referentiepunt van het sjabloon beschrijven, bijv. zwarte vierkanten op de hoeken en een zwart rechthoekig rotatiemarker ertussen.

### RotationMarkerPosition {#RotationMarkerPosition}
```
public RotationPointPosition RotationMarkerPosition
```


Gewenste positie van de rotatiemarker op het sjabloon. Indien niet ingesteld, wordt RotationPointPosition.TopRight1 gebruikt. De marker wordt getekend als een vijfde zwart rechthoek.
