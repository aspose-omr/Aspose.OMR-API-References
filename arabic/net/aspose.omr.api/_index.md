## الفئات

| الفئة | الوصف |
| --- | --- |
| [ImageCollection](./imagecollection) | مجموعة من الصور التي يمكن استخدامها لإنشاء القالب. مثال: [`Generate`](../aspose.omr.api/omrengine/generate) يمكن أن تكون مفيدة عندما لا تكون صورك موجودة في نظام الملفات. |
| [OmrEngine](./omrengine) | محرك OMR. يتعامل مع إنشاء القالب وفئات معالجة الصور ومكونات واجهة المستخدم الرسومية. |
| [Presets](./presets) | يوفر أوامر مختصرة لإنشاء سريع وتعريف نماذج OMR ذات تصميم مسبق. تتيح هذه الواجهة إنشاء النماذج دون الحاجة لتعلم لغات توصيف Aspose.OMR وتعريف النماذج دون ملف نمط. للتحكم المتقدم في تصميم النماذج والتعريف، استخدم [`OmrEngine`](../aspose.omr.api/omrengine) |
| [TemplateProcessor](./templateprocessor) | فئة لمعالجة القوالب والصور. كل مثال من هذه الفئة يعمل مع قالب OMR واحد. يمكنه التعرف على صور القالب المحدد في المُنشئ. |
