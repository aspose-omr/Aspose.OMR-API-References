Οι παγκόσμιες ρυθμίσεις για σημεία αναφοράς που ισχύουν για όλα τα στοιχεία της σελίδας.

Ρυθμίσεις που περιγράφουν τη συμπεριφορά για το Σημείο Αναφοράς του προτύπου π.χ. Μαύρα τετράγωνα στις γωνίες και μαύρο ορθογώνιο δείκτη περιστροφής μεταξύ τους.

```
public RotationPointPosition RotationMarkerPosition
```
