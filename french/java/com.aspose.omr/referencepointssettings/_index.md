Les paramètres globaux pour les points de référence applicables à tous les éléments de page.

Paramètres décrivant le comportement du point de référence du modèle, par ex. carrés noirs aux coins et rectangle noir du marqueur de rotation entre eux.

### RotationMarkerPosition {#RotationMarkerPosition}
```
public RotationPointPosition RotationMarkerPosition
```


Position souhaitée du marqueur de rotation sur le modèle. Si non définie, RotationPointPosition.TopRight1 sera utilisé. Le marqueur est dessiné comme un cinquième rectangle noir.
