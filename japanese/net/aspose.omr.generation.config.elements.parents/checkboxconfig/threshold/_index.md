## CheckBoxConfig.Threshold property

カスタム認識マーカー。 デフォルトでは 3% に設定されています。 チェック ボックスは、領域の 3% が満たされている場合にマークされていると見なされます。 他のしきい値にカスタマイズできます。 グローバルなしきい値には影響しません

```csharp
public int Threshold { get; set; }
```
