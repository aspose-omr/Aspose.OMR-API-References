La configuración global de puntos de referencia aplicable a todos los elementos de la página.

Configuración que describe el comportamiento del Punto de Referencia de la plantilla, p. ej., cuadrados negros en las esquinas y rectángulo negro del marcador de rotación entre ellos.

### RotationMarkerPosition {#RotationMarkerPosition}
```
public RotationPointPosition RotationMarkerPosition
```


Posición deseada del marcador de rotación en la plantilla. Si no se establece, se usará RotationPointPosition.TopRight1. El marcador se dibuja como un quinto rectángulo negro.
