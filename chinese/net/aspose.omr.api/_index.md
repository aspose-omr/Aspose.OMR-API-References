## 类

| 类 | 描述 |
| --- | --- |
| [ImageCollection](./imagecollection) | 可用于模板生成的图像集合。例如 [`Generate`](../aspose.omr.api/omrengine/generate) 当您的图像不在文件系统中时可能有用。 |
| [OmrEngine](./omrengine) | OMR 引擎。负责创建模板和图像处理类以及 GUI 组件。 |
| [Presets](./presets) | 提供快捷命令以快速生成和识别具有预定义设计的 OMR 表单。此 API 使得无需学习 Aspose.OMR 标记语言即可创建表单，并在没有模式文件的情况下进行表单识别。要对表单设计和识别进行高级控制，请使用 [`OmrEngine`](../aspose.omr.api/omrengine) |
| [TemplateProcessor](./templateprocessor) | 用于处理模板和图像的类。该类的每个实例都与单个 OMR 模板一起工作。它能够识别构造函数中指定的模板图像。 |
