## 类

| 类 | 描述 |
| --- | --- |
| [AnswerConfig](./answerconfig) | 表示 Answer 元素。可以作为子元素与 [`VerticalChoiceBoxConfig`](../aspose.omr.generation.config.elements.parents/verticalchoiceboxconfig) 一起使用 |
| [BlockConfig](./blockconfig) | 表示 Block 元素。 |
| [CheckBoxConfig](./checkboxconfig) | 表示 CheckBox 元素。 |
| [ContainerConfig](./containerconfig) | 表示 Container 元素。 |
| [PageConfig](./pageconfig) | 表示 Page 元素。每页一个。必须放置在 [`TemplateConfig`](../aspose.omr.generation.config/templateconfig) 内部。 |
| [ParagraphConfig](./paragraphconfig) | 表示 Paragraph 元素。Paragraph 可以由文本或带图像的文本组成。图像通过坐标 x、y 指定。 |
| [PositionedBlockConfig](./positionedblockconfig) | 分组元素。在画布上使用绝对坐标定位任何子元素 |
| [VerticalChoiceBoxConfig](./verticalchoiceboxconfig) | VerticalChoiceBox 由一个包含简短响应的 [`AnswerConfig`](../aspose.omr.generation.config.elements.parents/answerconfig) 元素和描述响应选项细节的 [`ContentConfig`](../aspose.omr.generation.config.elements/contentconfig) 元素组成。它还可以包含一个 [`WriteInConfig`](../aspose.omr.generation.config.elements/writeinconfig) 元素，用户可以在其中输入自己的答案。 |
