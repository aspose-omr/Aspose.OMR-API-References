## CheckBoxConfig.Threshold property

自定义识别标记。 默认设置为 3%。 如果复选框的 3% 区域被填充，则复选框将被视为已标记。 可以自定义为任何其他阈值。 不影响全局阈值

```csharp
public int Threshold { get; set; }
```
