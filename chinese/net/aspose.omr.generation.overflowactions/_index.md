## 课程

| 班级 | 描述 |
| --- | --- |
| [Clip](./clip/) | 溢出部分将被截断。 默认情况下，父元素将在页面上占用尽可能多的空间。 如果该大小不够 - 子元素将被截断。 |
| [NoClip](./noclip/) | 不会尝试控制溢出。 |
| [OverflowAction](./overflowaction/) | 控制子元素溢出父元素大小的操作。 等待实现会产生不同的结果。 |
| [Wrap](./wrap/) | 父元素将尝试在另一个 parents 中为溢出的子元素找到一个位置 |
