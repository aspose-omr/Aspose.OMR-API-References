Die globalen Einstellungen für Referenzpunkte, die für alle Seitenelemente gelten.

Einstellungen, die das Verhalten des Referenzpunkts der Vorlage beschreiben, z. B. schwarze Quadrate an den Ecken und ein schwarzes Rechteck als Rotationsmarker dazwischen.

### RotationMarkerPosition {#RotationMarkerPosition}
```
public RotationPointPosition RotationMarkerPosition
```


Gewünschte Position des Rotationsmarkers in der Vorlage. Wenn nicht festgelegt, wird RotationPointPosition.TopRight1 verwendet. Der Marker wird als fünftes schwarzes Rechteck gezeichnet.
