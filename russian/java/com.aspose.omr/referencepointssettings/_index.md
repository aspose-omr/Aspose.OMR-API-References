Глобальные настройки точек отсчёта, применимые ко всем элементам страницы.

Настройки, описывающие поведение точки привязки шаблона, например черные квадраты в углах и черный прямоугольник маркера вращения между ними.

### RotationMarkerPosition {#RotationMarkerPosition}
```
public RotationPointPosition RotationMarkerPosition
```
