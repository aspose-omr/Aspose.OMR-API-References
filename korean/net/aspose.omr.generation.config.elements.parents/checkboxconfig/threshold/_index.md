## CheckBoxConfig.Threshold property

사용자 지정 인식 마커. 기본적으로 3%로 설정됩니다. 영역의 3%가 채워진 경우 확인란이 표시된 것으로 간주됩니다. 다른 임계값으로 사용자 지정할 수 있습니다. 전역 임계값에 영향을 주지 않습니다

```csharp
public int Threshold { get; set; }
```
