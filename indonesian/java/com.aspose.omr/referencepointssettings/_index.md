Pengaturan global untuk titik referensi yang berlaku untuk semua elemen halaman.

Pengaturan yang menjelaskan perilaku untuk Titik Referensi pada templat, misalnya kotak hitam di sudut-sudut dan persegi panjang hitam penanda rotasi di antaranya.
