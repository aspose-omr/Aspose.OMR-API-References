## VerticalChoiceBoxConfig class

The VerticalChoiceBox consists of a [`AnswerConfig`](../answerconfig) element that contains a short response and [`ContentConfig`](../../aspose.omr.generation.config.elements/contentconfig) elements that describe the response option in detail. It can also contain a [`WriteInConfig`](../../aspose.omr.generation.config.elements/writeinconfig) element in which the user can enter his own answer.
